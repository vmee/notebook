# 演讲

## 本质
演讲的本质：关键内容、提炼抽象、排列组合

## 演讲结构
### 黄金圈法则结构：(((Why)How)What)
- Why 为什么： 目的、使命、信念
- How 怎么做： 过程、方法
- What 做什么： 结果
适用场景：各种

### PREP结构
- Point 观点
- Reason 理由
- Example 案例
- Point 观点
适用场景：各种


### 时间轴结构
- 过去、现在、未来
适用场景：总结类，CEO演讲

### 金字塔结构
问题 ->  原因 -> 对策 -> 结果
适用场景：公司会议、报告、发言
